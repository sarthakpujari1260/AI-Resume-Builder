🤖 Resumefy- AI Resume Builder

AI Resume Builder is a modern web application designed to simplify the resume creation process for students, fresh graduates, and professionals. Instead of struggling with formatting and wording, users can create polished resumes through an intuitive interface while leveraging AI-powered content assistance to generate professional summaries, project descriptions, skills, and experience statements.

The platform provides a complete resume-building workflow, including user authentication, profile management, dynamic resume sections, real-time preview, customizable templates, and PDF export functionality. It helps users create professional resumes quickly while maintaining consistency and clarity across all sections.

Tech Stack

Frontend: React.js · Tailwind CSS · React Router · Axios

Backend: Node.js · Express.js · JWT Authentication

Database: MongoDB

PDF Generation: React PDF / html2pdf (update according to your implementation)

AI Integration: Gemini API / OpenAI API (whichever you used)

Key Features
🤖 AI-powered resume content generation
📄 Multiple resume sections (Education, Skills, Experience, Projects, Certifications)
👤 Secure user authentication and profile management
✏️ Real-time resume editing and live preview
🎨 Professional resume templates
📥 PDF resume export
📱 Fully responsive user interface
💾 Save and update resumes anytime
⚡ Fast and intuitive resume-building workflow


The goal of AI Resume Builder is to help users create professional, well-structured resumes efficiently by combining modern web technologies with intelligent content generation, reducing the time and effort required to prepare job-ready resumes.
